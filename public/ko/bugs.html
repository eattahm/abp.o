애드블록 플러스의 버그가 아님

Please consider the following points before filing a bug report:

  • 광고가 차단되지 않음 : 필터의 문제이므로 그 문제를 보고해주십시오.
  • 웹사이트가 정상적으로 작동하지 않고 임시로 애드블록 플러스의 사용을 중지 하면 사이트의 문제가 고쳐짐 : 필터의 문제이므로 그 문제를 보고해주십시오.
  • 애드블록 플러스를 설치하기 전 또는 그 확장 프로그램을 제거했는데도 문제가 지속됨. : 해결방안은 다른 곳에서 찾아 보아야 합니다. If Adblock Plus fails to install on a supported browser, this is a browser issue and general troubleshooting instructions apply. If you are not using Firefox, see: Chrome, Opera, Safari, Internet Explorer

버그 보고 방법

Visit the forum for support or to report an issue.

버그 보고서에는 문제를 재현하기 위해 필요한 모든 정보가 포함되어 있어야 합니다.

  • The Adblock Plus version number, e.g. 1.8.12
  • The browser and browser version number, e.g. Chrome 43.0.2357.124
  • The computer operating system, e.g. Windows 8.1 64-bit
  • 문제와 관련된 자세한 설정사항 (사용 중인 구독 필터, 사용자 필터, 평범하지 않은 웹브라우저 설정, 설치한 다른 확장 프로그램)
  • 버그를 확인하기 위해 필요한 과정에 대한 명확한 설명
  • 어떠한 현상이 발생하고, 무엇이 비정상적인지에 대한 명확한 설명
