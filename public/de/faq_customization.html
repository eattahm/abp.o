Wie kriege ich das Adblock Plus Symbol in die Symbolleiste?

Normalerweise wird das Symbol automatisch bei der ersten Installation von Adblock Plus in die Symbolleiste eingefügt, aber vielleicht wurde es später entfernt. In Firefox kann man die Symbolleiste sehr einfach bearbeiten, indem man die Symbolleiste mit der rechten Maustaste anklickt und "Anpassen" auswählt. Es öffnet sich ein Fenster mit Symbolen, unter denen auch das Symbol für Adblock Plus zu finden sein sollte. Dieses muß man einfach mit der Maus auf den gewünschten Platz ziehen. Man kann die Symbole auch nach belieben umordnen, solange das Anpassungsfenster geöffnet ist.

Wie kann ich das Adblock Plus Symbol aus der Symbolleiste entfernen?

Sie mögen das Symbol nicht oder brauchen den Platz für wichtigere Sachen? Man kann ihn entfernen, indem man den kleinen Pfeil neben dem Symbol anklickt und im Menü Optionen / In Symbolleiste anzeigen auswählt. Das Menü Optionen findet man auch im Einstellungsdialog — falls man entscheidet, das Symbol wieder anzeigen zu lassen. Übrigens kann man auch die Symbolleisten-Anpassung benutzen, die in Firefox eingebaut ist.

Wie bekomme ich wieder ein grünes Symbol für ausgenommene Webseiten?

Falls Sie das Adblock Plus Symbol wieder so haben wollen, wie es vor Adblock Plus 1.3.5 war (grün wenn Adblock Plus auf einer Webseite deaktiviert ist, grau falls es überall deaktiviert ist), können Sie diesen User Style installieren (die Erweiterung Stylish wird benötigt).

Wie kann ich die Liste blockierbarer Elemente auf der linken/oberen Seite des Browsers anzeigen?

Wollen Sie die Liste blockierbarer Elemente wieder so haben, wie sie vor Adblock Plus 0.7.5 war? Das kann man mit diesem User Style bewerkstelligen (die Erweiterung Stylish wird benötigt). In dieser Form verschiebt das User Style die Liste an den linken Rand des Browserfesters, wie eine Sidebar. Man kann aber auch eine der markierten Zeilen des User Style entfernen, wenn man es hinzufügt — dann wird die Liste auf der rechten Seite beziehungsweise oben angezeigt.

Bemerkung: Die Liste der blockierbaren Elemente wird dadurch nicht zu einer richtigen Sidebar. So kann es zum Beispiel zusammen mit einer echten Sidebar wie Verlauf geöffnet sein.

Wie kann ich in der Statusleiste Text statt einem Symbol anzeigen lassen?

Frühere Versionen von Adblock Plus hatten in der Statusleiste das Wort "Adblock" stehen. Dieses Verhalten kann mit diesem User Style wiederhergestellt werden (die Erweiterung Stylishwird benötigt).

Wie kann ich das Adblock Plus Symbol in der Statusleiste an eine andere Stelle verschieben?

Adblock Plus 0.5 erlaubte es, das Symbol in der Statusleiste mit der Maus zu verschieben. Für die nachfolgenden Versionen wurde entschieden, daß diese Funktion nur selten benutzt wird und viel besser von einer spezialisierten Erweiterung wie z.B. Organize Status Bar realisiert werden kann.

Wie entferne ich Adblock Plus aus allen Browser-Menüs?

Um Adblock Plus aus dem "Extras"-Menü und dem Kontextmenü zu entfernen, sollten Sie Customizations for Adblock Plus-Erweiterung installieren. Gehen Sie nach der Installation zum Add-ons-Mananger und klicken Sie bei Customizations auf "Einstellungen". Setzen Sie dann einen Haken bei "Adblock Plus aus Browser-Menüs entfernen".

Wie ändere ich die Symbole, die Adblock Plus verwendet?

Falls Sie die Symbole bevorzugen, die vor Adblock Plus 0.7.2 verwendet wurden (die Schabe), können Sie diesen User Style anwenden, um sie wiederherzustellen (am einfachsten geht das mit Stylish). Falls Sie aber bessere Symbole haben, können Sie diese Schablone für ein eigenes Stylesheet verwenden:

@namespace url("http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul");

@-moz-document url("chrome://browser/content/browser.xul"),
      url("chrome://navigator/content/navigator.xul"),
      url("chrome://messenger/content/messenger.xul"),
      url("chrome://messenger/content/messageWindow.xul"),
      url("chrome://midbrowser/content/midbrowser.xul"),
      url("chrome://emusic/content/startup.xul"),
      url("chrome://webrunner/content/webrunner.xul"),
      url("chrome://global/content/customizeToolbar.xul") {

  toolbar[iconsize="small"] #abp-toolbarbutton,
  #PersonalToolbar #abp-toolbarbutton,
  #abp-status {
    list-style-image: url("file:///c:/icons/abp-16x16.png") !important;
  }

  #abp-toolbarbutton {
    list-style-image: url("file:///c:/icons/abp-24x24.png") !important;
  }
}

Hier wird davon ausgegangen, dass sich die Symbole in dem Verzeichnis c:\icons befinden, Sie sollten die Adressen also gegebenenfalls anpassen. Um dieses Stylesheet anzuwenden, benutzen Sie entweder Stylish oder bearbeiten die Datei chrome/"userChrome.css":http://www.firefox-browser.de/wiki/UserChrome.css in Ihrem Profilverzeichnis und fügen diesen Code hinzu. Falls userChrome.css noch nicht existiert, müssen Sie diese Datei erstellen.

Anmerkung: Die Änderungen werden erst nach einem Browserneustart übernommen.

Wie ändere ich das Tastenkürzel für ... / definiere ein Tastenkürzel zum Deaktivieren von Adblock Plus?

Die Tastaturkürzel in Adblock Plus werden über drei versteckte Einstellungen festgelegt, die über about:config zugängig sind. extensions.adblockplus.sidebar_key steht fürs Öffnen/Schließen der blockierbaren Elemente, extensions.adblockplus.settings_key fürs Öffnen der Einstellungen und extensions.adblockplus.enable_key fürs Aktivieren/Deaktivieren von Adblock Plus. Sie können diese nach belieben ändern.

Anmerkung: Die Änderungen werden erst nach einem Browserneustart übernommen.
