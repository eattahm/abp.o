Démarrage

Adblock Plus va automatiquement sélectionner un abonnement à une liste de filtre qui vous est adapté. Vous pouvez toujours sélectionner un autre abonnement dans la liste.

Adblock Plus va fonctionner comme un service en arrière-plan et son status sera affiché dans la zone de notification.

Sélectionner un abonnement

Vous pouvez à tout moment changer d'abonnement en le sélectionnant dans la liste suivante :

Dès que vous sélectionnez un autre abonnement, celui-ci est automatiquement téléchargé et activé. Les abonnements sont par défaut mis à jour automatiquement. Deux options sont disponibles dans les paramètres avancés :

  • Manuellement — l'abonnement ne se met à jour que lorsque vous appuyez sur le bouton de rafraichissement ().
  • Au démarrage — l'abonnement est automatiquement mis à jour lorsque vous démarrez Adblock Plus.

Paramétrer manuellement le proxy

Les versions d'Android inférieures à 3.1 ne permettent pas nativement de modifier les paramètres de proxy, mais plusieurs constructeurs ont ajouté cette possibilité à leur appareil. Si vous possédez un appareil non-rooté dont la version d'Android est inférieure à 3.1, il vous est toujours possible de configurer le proxy manuellement.

En fonction de l'appareil, la configuration du proxy se trouve à différents endroits. Si votre appareil se trouve dans la liste suivante, merci de suivre les instructions qui leur sont spécifiques :

Instructions génériques

If your device isn't in the list, the most common place for the proxy settings is in the advanced WiFi settings.

Android 4.x (Ice Cream Sandwich, Jelly Bean, KitKat) devices

Run Settings application. Select Wi-Fi. The list of available Wi-Fi networks will open:

Connect to the desired network (access point) if not already. Long press on connected network, context menu will appear. Select Modify network config:

In opened dialog mark Show advanced options check box:

Scroll to proxy settings, select Manual, proxy options will appear:

Set Proxy to localhost and Port to the number specified by Adblock Plus in the notification message:

Android 2.3 (Gingerbread) devices

Go to the WiFi settings and select Advanced settings from the menu:

Cherchez l'hôte de proxy et la configuration du port :

Mettez localhost comme hôte de proxy et le numéro affiché par Adblock Plus pour le port :
