我（不小心）屏蔽了一些东西，如何才能恢复显示呢？

对于 Firefox：

  1. 点击 Firefox >> 附加组件（ 对于 Mac OS X / Linux，点击菜单栏的"工具" >> "附加组件"），打开 Firefox 附加组件管理器。
  2. 点击"扩展"，找到 Adblock Plus，进入"选项"并点击"过滤规则首选项..."。
  3. 切换到上方的"自定义过滤规则"标签页，这里会显示您的自定义过滤规则分组。
  4. 在某个分组上点击右键，选择"显示/隐藏过滤规则"以显示您的自定义规则。
  5. 在右侧窗口移除相应的过滤规则，这样就可以重新显示您屏蔽的元素。

对于 Chrome：

  1. 点击 Chrome 的菜单按钮，找到"工具"并点击"扩展"。
  2. 找到 Adblock Plus，并点击扩展描述下方的"选项"按钮。
  3. 切换到"添加自定义过滤规则"标签页，在此您可以移除自定义的过滤规则。

对于 Opera：

  1. 点击"菜单"按钮（对于 Mac OS X / Linux，点击"工具"），选择"扩展" >> "管理扩展"。
  2. 找到 Adblock Plus 并点击右侧的小工具按钮，接着点击"首选项"。
  3. 切换到"添加自定义过滤规则"标签页，在此您可以移除自定义的过滤规则。

我装了 Adblock Plus，但怎么还会看到广告？

请先确认 Adblock Plus 已经启用

对于 Firefox：

点击 Firefox >> 附加组件（ 对于 Mac OS X / Linux，点击菜单栏的"工具" >> "附加组件"），打开 Firefox 附加组件管理器。接着点击"扩展"，找到 Adblock Plus，进入"选项" ，确认已取消勾选"完全禁用"这项。

对于 Chrome：

点击 Chrome 的菜单按钮，选择"工具"并点击"扩展"。找到 Adblock Plus 并确保已勾选右侧的"已启用"字样。

对于 Opera：

点击"菜单"按钮（对于 Mac OS X / Linux，点击"工具"），选择"扩展" >> "管理扩展"。找到 Adblock Plus，确认扩展未呈现灰色状态并已启用。
