Preferencias de Adblock Plus

La siguiente tabla detalla y explica todas las preferencias utilizadas por Adblock Plus, que pueden visualizarse y modificarse a través de about:config.

Preferencia Valor por defecto Comentarios
Opciones visibles
extensions. adblockplus. enabled true Valor de la opción "Activar Adblock Plus"
extensions. adblockplus. frameobjects true Valor de la opción "Mostrar pestañas en Flash y Java"
extensions. adblockplus. savestats false Valor de la opción "Contar usos de filtros"
extensions. adblockplus. showinstatusbar false Valor de la opción "Mostrar en la barra de estado"
Opciones ocultas
extensions. adblockplus. clearStatsOnHistoryPurge true Si tiene el valor true, al borrar el historial de navegación se borrarán también los resultados de los contadores de uso de los filtros de Adblock Plus. Tenga en cuenta que los resultados de los contadores de uso a menudo permiten deducir qué sitios visitó el usuario, cuando y con qué frecuencia.
extensions. adblockplus. composer_default 2 Determina qué sugerencia debe estar seleccionada por defecto en la ventana de elaboración de filtros:

0 — modelo personalizado
1 — sólo dirección exacta
2 — todo el directorio
3 — todo el sitio web
extensions. adblockplus. data_directory adblockplus Unused in Adblock Plus 2.9 and higher.
extensions. adblockplus. defaultstatusbaraction 0 Determina la acción a ejecutar cuando se pulsa el icono de la barra de estado. Puede ver una lista de posibles valores en defaulttoolbaraction.
extensions. adblockplus. defaulttoolbaraction 0 Determina la acción a ejecutar cuando se pulsa el icono de la barra de herramientas. Los valores posibles son:

0 — mostrar menú
1 — abrir/cerrar ventana de elementos bloqueables
2 — mostrar la ventana de Preferencias
3 — activar/desactivar Adblock Plus
cualquier otro — no hacer nada
extensions. adblockplus. enable_key vacío Determina el atajo de teclado a utilizar para activar/desactivar Adblock Plus. Para una descripción de sus posibles valores puede consultar sidebar_key.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.
extensions. adblockplus. fastcollapse false If set to true blocked elements will no longer be hidden. This might have the effect that placeholders will show up for blocked images for example, unless hidden by other means. This functionality was available as "Hide placeholders of blocked elements" before Adblock Plus 2.6.7.
extensions. adblockplus. filters_key Accel Shift E, Accel Shift F, Accel Shift O Define el atajo de teclado a usar para abrir la ventana de preferencias de filtros. Para ver una descripción de los posibles valores puede consultar sidebar_key.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.
extensions. adblockplus. flash_scrolltoitem true Determina si los "bordes de elementos Flash" en la lista de la ventana de elementos bloqueables deben intentar desplazar la ventana de forma que el elemento sea visible.
extensions. adblockplus. patternsbackupinterval 24 Intervalo mínimo entre dos copias de seguridad sucesivas de patterns.ini (en horas)
extensions. adblockplus. patternsbackups 5 Número de copias de seguridad a mantener para el archivo "patterns.ini". Si el valor es 0 se deshabilitarán las copias de seguridad.
extensions. adblockplus. patternsfile vacío Nombre del archivo a utilizar para guardar la lista de filtros. Puede ser un direccionamiento absoluto o un direccionamiento relativo del archivo respecto al directorio de perfil del usuario.
extensions. adblockplus. please_kill_startup_performance false Unused in Adblock Plus 2.9 and higher.
extensions. adblockplus. previewimages true Determina si el aviso emergente (tooltip) en la lista de la ventana de elementos bloqueables debe mostrar la imagen representada por el elemento.
extensions. adblockplus. sendReport_key vacío Determina el atajo de teclado a utilizar para enviar informes de errores. Para ver la descripción de sus posibles valores consulte sidebar_key.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.
extensions. adblockplus. sidebar_key Accel Shift V, Accel Shift U Determina el atajo de teclado a usar para abrir la ventana de elementos bloqueables. Esta preferencia puede estar vacía o tener el siguiente formato modificador1 modificador2 tecla. Los modificadores reconocidos son: Shift, Accel (tecla de aceleración preterminada, como Ctrl en Windows, Comando en Mac OS X), Ctrl/Control, Alt, Meta. La tecla puede ser una letra o una tecla especial como F7 o FLECHA DERECHA (puede buscar en la interfaz de nsIDOMKeyEvent la lista de teclas especiales). A partir de Adblock Plus 1.3.5 se pueden especificar varios atajos de teclado separado por comas, se utilizará el primero de la lista que se halle disponible.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.
extensions. adblockplus. subscriptions_autoupdate true Define si los filtros de las suscripciones se actualizarán automáticamente.

Nota: No es recomendable desactivar las actualizaciones.
extensions. adblockplus. subscriptions_exceptionscheckbox true Si tiene el valor true, la lista de filtros para publicidad no intrusiva tendrá un tratamiento especial y se mostrará como una opción seleccionable en la interfaz de usuario.
Acciones memorizadas
extensions. adblockplus. blockableItemsSize {width: 200, height: 200} Tamaño guardado de la lista de elementos bloqueables (la predisposición de la anchura sólo es aplicable cuando se muestra como panel lateral).
extensions. adblockplus. currentVersion 0.0 Guarda la última versión utilizada de Adblock Plus, esto desencadenará las actuaciones necesarias en una actualización o primera instalación
extensions. adblockplus. detachsidebar false Guarda el estado de la ventana de elementos bloqueables — el valor true corresponde a separada (no fija en el lateral)
extensions. adblockplus. hideContributeButton false Si tiene el valor true significa que el usuario a elegido ocultar el botón de Contribuir.
extensions. adblockplus. notifications_ignoredcategories [] List of notification categories that the user opted out of.
extensions. adblockplus. notifications_showui false Determines whether notification management UI should be displayed (only if the user opted out of notifications before).
extensions. adblockplus. recentReports [] Datos codificados en formato JSON sobre los informes de errores enviados recientemente (dirección del informe, página web implicada y fecha/hora de envío)
Preferencias internas (no las cambie si no está realmente seguro de lo que está haciendo)
extensions. adblockplus. analytics { trustedHosts: ["adblockplus.org", "notification.adblockplus.org", "easylist-downloads.adblockplus.org"] } Contains analytics data, for example the version of the first fetched resource, and a list of trusted hosts which that data can be shared with.
extensions. adblockplus. documentation_link   Dirección de adblockplus.org a abrir en los enlaces de documentación. Redireccionará a la ubicación real de la documentación en función del idioma utilizado.
extensions. adblockplus. notificationdata {} Datos de las notificaciones de emergencia actuales.
extensions. adblockplus. notificationurl   Ubicación del archivo que contiene las notificaciones de emergencia.
extensions. adblockplus. report_submiturl   Dirección de adblockplus.org a la que enviar informes de errores.
extensions. adblockplus. subscriptions_antiadblockurl   Location of the subscription that hides anti adblock messages.
extensions. adblockplus. subscriptions_exceptionsurl   Ubicación de la lista de filtros usada para permitir publicidad no intrusiva.
extensions. adblockplus. subscriptions_exceptionsurl_privacy   Location of the filter list used to allow only nonintrusive ads that are privacy-friendly.
extensions. adblockplus. subscriptions_fallbackerrors 5 Establece el número de intentos fallidos de descarga de una suscripción que deben producirse antes de consultar el script de dirección alternativa (consulte subscriptions_fallbackurl) para obtener la nueva dirección de la suscripción. Tenga en cuenta que Adblock Plus siempre intentará descargar de nuevo cuando haya pasado un día desde un intento fallido de descarga.
extensions. adblockplus. subscriptions_fallbackurl   Dirección del script a consultar si no se consigue descargar una suscripción un determinado número de veces (consulte subscriptions_fallbackerrors). Se sustituye %s por la dirección de dicha suscripción. El script debe imprimir algo similar a "301 http://example.com/new_list.txt" si se conoce la nueva ubicación de la suscripción.
extensions. adblockplus. subscriptions_listurl   Dirección de la que descargar la lista completa de suscripciones de filtros.
extensions. adblockplus. whitelistschemes about chrome file irc moz-extension moz-safe-about news resource snews x-jsd addbook cid imap mailbox nntp pop data javascript moz-icon Lista de configuraciones de URL que no deben ser bloqueadas (se aplica tanto a documentos completos como a ubicaciones de elementos)
