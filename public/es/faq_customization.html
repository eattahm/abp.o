¿Cómo pongo el icono de Adblock Plus en la barra de herramientas?

Normalmente el icono se añade automáticamente la primera vez que instala Adblock Plus, pero quizás se eliminó de algún modo después. En Firefox puede personalizar la barra de herramientas haciendo clic derecho en la barra de herramientas y seleccionando "Personalizar...". Se abrirá una ventana con iconos, y entre ellos debe estar el de Adblock Plus. Arrástrelo con el ratón hasta la barra de herramientas y colóquelo donde desee. Puede cambiar de lugar los iconos de la barra arrastrándolos mientras sigue abierta la ventana de personalización.

¿Cómo puedo quitar el icono de Adblock Plus de la barra de herramientas?

Si no le gusta el icono o necesita el espacio para cosas más importantes, pulse sobre la flecha al lado del icono, seleccione menú "Opciones" y luego "Mostrar en la barra de herramientas". También puede acceder al menú de Opciones en la ventana de Preferencias — por si acaso decide más tarde reponer el icono. Por cierto, puede utilizar la personalización de barras de herramientas que trae de serie Firefox.

¿Cómo puedo hacer que el icono se vuelva verde en los sitios web de la lista blanca?

Si quiere que el icono de Adblock Plus vuelva a comportarse como antes de Adblock Plus 1.3.5 (verde cuando Adblock Plus está desactivado en un sitio web, gris si está desactivado de forma global), puede instalar este estilo de usuario (necesitará la extensión Stylish).

¿Cómo consigo que la lista de elementos bloqueables se muestre en el lateral/en la parte superior?

¿Prefiere que la lista de elementos bloqueables esté en el lado izquierdo del navegador como ocurría antes de Adblock Plus 0.7.5? Puede hacer que sea así con este estilo de usuario (necesitará la extensión Stylish). De este modo el estilo se asegurará de que la lista se muestre en el lado izquierdo de la ventana del navegador, como un panel lateral. También puede eliminar una de las líneas marcadas cuando aparezca el mensaje para añadir el estilo — en ese caso la lista de elementos bloqueables se mostrará en el lado derecho o en la parte superior.

Nota: Esto no hace que la lista de elementos bloqueables sea un verdadero panel lateral. Se puede abrir al mismo tiempo que un verdadero panel lateral como el del Historial.

¿Cómo puedo hacer para que aparezca texto en la barra de estado en vez de un icono?

Las versiones anterior de Adblock Plus mostraban la palabra "Adblock" en la barra de estado. Este comportamiento puede reponerse con este estilo de usuario (necesitará la extensión Stylish).

¿Cómo puedo mover el icono de Adblock Plus desde la barra de estado a otro lugar?

Adblock Plus 0.5 permitía arrastrar el icono dentro de la barra de estado. En las siguientes versiones se decidió que esta funcionalidad se usa raramente y que no debería ir integrada. Una extensión como Organize Status Bar puede hacer eso mucho mejor.

¿Cómo elimino Adblock Plus de todos los menús del navegador?

Para ocultar las entradas de Adblock Plus del menú Herramientas y del menú contextual tendrá que instalar la extensión Customizations for Adblock Plus. Después de instalarla, búsquela en el Administrador de complementos y pulse el botón de Opciones de la extensión. Marque la opción de "Eliminar Adblock Plus de los menús del navegador".

¿Cómo cambio los iconos que muestra Adblock Plus?

Si prefiere los iconos utilizados antes de Adblock Plus 0.7.2 (la cucaracha), puede usar este estilo de usuario para reponerlo (se recomienda usarlo con Stylish). Pero si tienen un icono mejor, puede crear su propio estilo utilizando esta plantilla:

@namespace url("http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul");

@-moz-document url("chrome://browser/content/browser.xul"),
      url("chrome://navigator/content/navigator.xul"),
      url("chrome://messenger/content/messenger.xul"),
      url("chrome://messenger/content/messageWindow.xul"),
      url("chrome://midbrowser/content/midbrowser.xul"),
      url("chrome://emusic/content/startup.xul"),
      url("chrome://webrunner/content/webrunner.xul"),
      url("chrome://global/content/customizeToolbar.xul") {

  toolbar[iconsize="small"] #abp-toolbarbutton,
  #PersonalToolbar #abp-toolbarbutton,
  #abp-status {
    list-style-image: url("file:///c:/icons/abp-16x16.png") !important;
  }

  #abp-toolbarbutton {
    list-style-image: url("file:///c:/icons/abp-24x24.png") !important;
  }
}

Se asume que las imágenes (una con tres iconos de 16×16 y otra con tres iconos de 24×24 apilados verticalmente) están ubicados en el directorio c:\icons, si no es así tiene que cambiar la dirección. Para aplicar este estilo puede usar Stylish o editar el archivo chrome/userChrome.css en su directorio de perfil (¿dónde está mi directorio de perfil?) y añadir este código. Si no existe el archivo userChrome.css tendrá que crearlo.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.

¿Cómo puedo cambiar el atajo de teclado para... / añadir un atajo de teclado para desactivar Adblock Plus?

Existen tres preferencias ocultas a las que se puede llegar a través de about:config y que definen los atajos de teclado para Adblock Plus extensions.adblockplus.sidebar_key (abrir/cerrar la ventana de elementos bloqueables), extensions.adblockplus.settings_key (abrir el diálogo de Preferencias), y extensions.adblockplus.enable_key (activar/desactivar Adblock Plus). Puede cambiar los tres como desee.

Nota: Los cambios no surtirán efecto hasta que no reinicie su navegador.
